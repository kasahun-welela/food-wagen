# Food‑Wagen  
A food browsing web application built with Next.js and TypeScript.

## Project Overview  
Food‑Wagen is a modern web app built using the Next.js framework (TypeScript) that allows users to browse food items.  
Key highlights:  
- Built with Next.js (app directory) + TypeScript  
- TailwindCSS  for styling  
- Deployed (or ready to deploy) on Vercel  <a href="https://food-wagen-three.vercel.app/"> Link </a>

##  Tech Stack  
- Framework: Next.js (TypeScript)  
- Styling: TailwindCSS 
- Code linting & formatting: ESLint (configured via `eslint.config.mjs`)  
- Project structure follows Next.js app folder conventions  
- Deployment target: Vercel   

##  Getting Started  
To run this project locally, follow the steps:

### Prerequisites  
Make sure you have installed:  
- Node.js (latest LTS recommended)  
- npm preferred package manager  

### Installation  
1. Clone the repository:  
   ```bash
   git clone https://github.com/kasahun‑welela/food‑wagen.git
   cd food‑wagen
2. Install dependencies:
   ```bash
      npm install
3. Run the development server: 
      ```bash
      npm run dev
4. Open your browser and go to <a href="http://localhost:3000"> http://localhost:3000 </a>
 to view the app.
